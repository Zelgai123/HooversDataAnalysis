## Installation
Please execute the following commands in a bash to start the web app in your local development environment.

```
$ git clone https://github.com/Zelgai123/HooversDataAnalysis.git
$ cd CovidTracker
$ npm install
$ npm run start
$ open http://localhost:3000/ 
```
